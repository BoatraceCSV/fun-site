# ボートレース ドメイン知識

fun-site が予想ロジックの土台にしているボートレース固有の知識をまとめる。

## 1. レース展開予想に使うデータと優先度

### 必須（朝の初回ビルドから利用可能）

データソース: BoatraceCSV `programs/title` + `programs/race_cards`

- `programs/title`: レース名・グレード・締切時刻などのメタ情報
- `programs/race_cards`: 選手・モーター・成績の出走表本体

| データ項目 | 説明 | BoatraceCSV カラム |
|---|---|---|
| 枠番（艇番） | 1〜6 号艇。枠番＝コースとほぼ一致（1 号艇の 1 コース取得率 ~98%） | `艇番` |
| 選手級別 | A1, A2, B1, B2 の 4 段階 | `級別` |
| 全国勝率 | 選手の全国勝率 | `全国勝率` |
| 全国2連対率 | 選手の全国2連対率 | `全国2連対率` |
| 当地勝率 | 当該競艇場での勝率 | `当地勝率` |
| 当地2連対率 | 当該競艇場での2連対率 | `当地2連対率` |
| モーター2連対率 | 現在のモーターの成績 | `モーター2連対率` |
| ボート2連対率 | 現在のボートの成績 | `ボート2連対率` |
| 選手体重 | 軽い方が有利（特にモーニング・ナイター） | `体重` |
| 今節成績 | 節間の調子（最大 6 レース分） | `今節成績1`〜`今節成績6` |
| 全国平均ST | 選手の全国平均スタートタイミング | `全国平均ST` |

### AI 総合評価

データソース: BoatraceCSV `estimate/index`

予想者の `componentKeys` に応じた要素（例: 本命予想は 枠番 / 選手 / モーター / 展示 / 気象)の
寄与pt を合計したものが「強さpt」。コース予想 (v6_course) は枠番の代わりに
**コース**要素（場×レース番号×コース別強度、`N枠_寄与_コースpt`)を使う。

| データ項目 | 説明 | BoatraceCSV カラム |
|---|---|---|
| 状態 | `daily`（直前情報未取得時）/ `realtime`（直前情報反映後） | `状態` |
| 枠番 pt 寄与 | 枠番要素の寄与スコア | `N枠_寄与_枠番pt` |
| コース pt 寄与 | 場×レース番号別コース強度の寄与スコア（v6_course のみ。枠番の代替） | `N枠_寄与_コースpt` |
| 選手 pt 寄与 | 選手要素の寄与スコア | `N枠_寄与_選手pt` |
| モーター pt 寄与 | モーター要素の寄与スコア | `N枠_寄与_モーターpt` |
| 展示 pt 寄与 | 展示要素の寄与スコア（state=daily では暫定値） | `N枠_寄与_展示pt` |
| 気象 pt 寄与 | 気象要素の寄与スコア（state=daily では暫定値） | `N枠_寄与_気象pt` |
| 強さpt | 採用要素の寄与合計 | `N枠_強さpt` |

`state=daily` のレースでは展示・気象が暫定値（寄与 0）なので、UI 側で展示・気象セグメントを除外し 3 要素のみ表示する。

### 直前情報

データソース: BoatraceCSV `previews/stt`（締切 5 分前以降に順次公開）

| データ項目 | 説明 | 備考 |
|---|---|---|
| 進入コース | 実際のコース取り（枠なり or 前付け） | 枠番と異なる場合がある |
| スタート展示 | 試走でのスタートタイミング | 実際のスタート力の指標 |

未取得のレースは進入コース＝枠番、スタートタイミング＝全国平均ST にフォールバックする。

### レース結果

データソース: BoatraceCSV `results/realtime`（当日確定直後に追記）

| データ項目 | 説明 |
|---|---|
| 着順 | 1〜6 着 |
| 決まり手 | 逃げ / 差し / まくり / まくり差し / 抜き / 恵まれ |
| ST | 各艇の実測スタートタイミング |
| 天候・風向・風速・波高 | 着順と並んで配信 |

K-file 由来の翌日確定 `results/daily` は本サイトでは使わない。

### 払戻金

データソース: BoatraceCSV `results/payouts`（当日確定直後に追記）

| データ項目 | 説明 |
|---|---|
| 単勝 / 複勝 | 1着艇 / 上位 2-3 着艇の払戻金 |
| 2連単 / 2連複 | 組番・払戻金・人気 |
| 拡連複 (3スロット固定) | 1-2着 / 1-3着 / 2-3着 の各ペア。販売されないレースは null |
| 3連単 / 3連複 | 組番・払戻金・人気 |

`results/realtime` (着順) と独立に追記される。**3連単 戦略の的中率・回収率算定**に使う。

### 利用していないデータ

オッズ・実測展示タイム・チルト角度・水面コンディションの詳細などは、現状の BoatraceCSV では取得できないか、本サイトでは使っていない。

## 2. 決まり手パターン

1 マーク（第 1 ターンマーク）の攻防でほぼ勝敗が決まる。

| 決まり手 | 説明 | 発生条件 |
|---|---|---|
| イン逃げ | 1 コースがそのまま先行して逃げ切り | 1 コースが A1 級、モーター好調、追い風 |
| 差し | 内側から 2〜3 コースが差す | 1 コースがターンで膨らんだ場合、2 コースの差し巧者 |
| まくり | 外側（主に 3〜4 コース）から一気にまくる | カド（4 コース）のスタートが早い、向かい風 |
| まくり差し | 外からまくりつつ内を差す | 高度な技術が必要、A1 級の実力者に多い |
| 抜き | 2 周目以降に逆転 | モーター性能差が大きい場合 |
| 恵まれ | 前の艇の失敗で繰り上がり | 転覆・落水・エンスト |

### スリット隊形

| 隊形 | 特徴 | よくある展開 |
|---|---|---|
| 横一線 | 全艇がほぼ同時にスリットを通過 | イン逃げ or 差し |
| 内凹み | インがスタート遅れ、外が出る | まくり or まくり差し |
| 外凹み | アウトがスタート遅れ | イン逃げ確率 UP |

### 艇番の色

公式の艇色対応。スタート予想 SVG・グラフの色分けに使用。

| 艇番 | 色 |
|---|---|
| 1 | 白 |
| 2 | 黒 |
| 3 | 赤 |
| 4 | 青 |
| 5 | 黄 |
| 6 | 緑 |

実装は [`packages/shared/src/constants/boat-colors.ts`](../packages/shared/src/constants/boat-colors.ts)。

## 3. 競艇場の特性

全 24 場のマスタは [`packages/shared/src/constants/stadiums.ts`](../packages/shared/src/constants/stadiums.ts)。

### イン天国（1 コース有利）

| 場 | 1 コース 1 着率の目安 | 特徴 |
|---|---|---|
| 大村 | ~65–70% | 全国 1 位。企画レースが多い |
| 徳山 | ~60% | コース広い、待機行動短い |
| 芦屋 | ~58% | サンライズV戦のイン勝率が高い |
| 下関 | ~57% | コース形状でイン有利 |

### 波乱場（穴狙い向き）

| 場 | 1 コース 1 着率の目安 | 特徴 |
|---|---|---|
| 戸田 | ~43% | 全国最狭水面（107.5m） |
| 江戸川 | ~44% | 唯一の河川コース、風と潮で大荒れ |
| 平和島 | ~45% | 風向・風速が不安定 |
| びわこ | ~47% | 高地で気圧低い、モーター出力↓ |

## 4. ページ表示上のフォールバック

直前情報や AI 評価の状態に応じて UI を切り替える。

| 状況 | 表示 |
|---|---|
| `previews/stt` 取得済み | 実測の進入コース・スタート展示を表示 |
| `previews/stt` 未取得 | 進入コース＝枠番、ST＝全国平均ST で仮表示（v5_slit の推定 ST 版は AI 推定 ST） |
| `estimate/index` `状態=daily` | 寄与pt から展示・気象を除外、3 要素のみ表示 |
| `estimate/index` `状態=realtime` | 5 要素すべて表示 |
| `results/realtime` 取得済み | レース詳細ページに結果セクションを表示 |
| `results/realtime` 未取得 | 結果セクションは非表示 |
| `results/payouts` 取得済み | レース詳細ページに「もし買ったら」セクション、トップに当日サマリーを表示 |
| `results/payouts` 未取得 | 両セクションともそのレースは集計から除外 |

予測 ST は既定で全国平均 ST。**スリット予想 (v5_slit) のみ** AI 推定 ST
(BoatraceCSV `estimate/racer_st`。実測 ST 履歴の時間減衰平均 + コース・F 補正)を
優先し、未生成時は全国平均 ST にフォールバックする
(`packages/shared/src/utils/one-mark-distance.ts` の `predictedST` / opt-in は
`PredictorSpec.useEstimatedST`)。
全国平均ST が 0.00(公表実績なし。新人・長期離脱明け等)の艇は、スタート予想図の描画と
1 マーク走行距離計算の両方で `NO_RECORD_ST_FALLBACK`(0.25 秒)に補完する(`effectiveAvgST`)。
補完しない場合 0.00 が「最速スタート扱い」になるため。
